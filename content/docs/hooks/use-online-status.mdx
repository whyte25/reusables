## Usage

The hook provides real-time network status monitoring with connection quality metrics.

```ts
import useOnlineStatus from "@/hooks/use-online-status"
```

```ts
const {
  isOnline,
  lastChanged,
  connectionQuality,
  latency,
  isRetrying,
  checkConnectionNow,
  retryConnection,
} = useOnlineStatus()
```
