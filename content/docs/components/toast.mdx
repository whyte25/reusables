## Styling

You can customize the appearance using the classNames prop:

```tsx
<ToastProvider
  classNames={{
    error: "bg-red-100 text-red-900",
    success: "bg-green-100 text-green-900",
    title: "font-bold",
    description: "text-sm",
    closeButton: "hover:bg-gray-200",
    containerClassName: "gap-2",
  }}
>
  {children}
</ToastProvider>
```
