---
title: Compact PDF Viewer
description: A compact PDF viewer component with basic navigation controls and clean interface.
---

## Features

- Basic page navigation
- Page number indicator
- Responsive design
- Dark mode support
- Clean minimal interface

## Usage

```tsx
import { CompactViewer } from "@/components/pdf-viewers/compact-viewer"

export default function PDFViewer() {
  return <CompactViewer url="path/to/your.pdf" />
}
```
