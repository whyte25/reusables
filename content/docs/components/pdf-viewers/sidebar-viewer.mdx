---
title: Sidebar PDF Viewer
description: A PDF viewer with a collapsible sidebar for page navigation and table of contents.
---

## Features

- Collapsible sidebar
- Page navigation
- Table of contents
- Mobile responsive
- Dark mode support

## Usage

```tsx
import { SidebarViewer } from "@/components/pdf-viewers/sidebar-viewer"

export default function PDFViewer() {
  return <SidebarViewer url="path/to/your.pdf" />
}
```
