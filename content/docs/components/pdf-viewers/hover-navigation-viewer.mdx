---
title: Hover Navigation PDF Viewer
description: A PDF viewer with hover-activated navigation controls.
---

## Features

- Hover-activated navigation
- Clean interface
- Page counter
- Responsive design
- Dark mode support

## Usage

```tsx
import { HoverNavigationViewer } from "@/components/pdf-viewers/hover-navigation-viewer";

export default function PDFViewer() {
  return <HoverNavigationViewer url="path/to/your.pdf" />;
}
```
