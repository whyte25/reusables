---
title: Thumbnail PDF Viewer
description: A PDF viewer with thumbnail previews for easy page navigation.
---

## Features

- Thumbnail previews
- Page navigation
- Responsive layout
- Dark mode support
- Smooth scrolling

## Usage

```tsx
import { ThumbnailViewer } from "@/components/ui/pdf-viewers/thumbnail-viewer";

export default function PDFViewer() {
  return <ThumbnailViewer url="path/to/your.pdf" />;
}
```
