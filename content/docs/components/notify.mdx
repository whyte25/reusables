## Toast Provider

Override default toast behavior using the ToastProvider.

```tsx
<ToastProvider
  position="bottom-right"
  duration={4000}
  closable={true}
  classNames={{
    error: "bg-red-100 text-red-900",
    success: "bg-green-100 text-green-900",
    title: "font-bold",
    description: "text-sm",
    closeButton: "hover:bg-gray-200",
    containerClassName: "gap-2",
  }}
  preventDuplicates={true}
  maxToast={4}
>
  {children}
</ToastProvider>
```

## Toast Push Method

For complete control over toast configuration, use the `toast.push` method:

```tsx
toast.push({
  title: "Custom Toast",
  status: "info", // 'error' | 'warning' | 'success' | 'info' | 'default' | 'loading'
  position: "top-right",
  duration: 5000,
  description: "This is a custom toast",
  closable: true,
  hideProgressBar: false,
  classNames: {
    title: "font-bold",
    description: "text-sm",
    closeButton: "hover:bg-gray-200",
  },
})
```
