## Usage

Basic usage with default options:

```tsx
const time = timeAgo(date)
// "2 hours ago"
```

Enhanced formatting with options:

```tsx
const time = formatTimeAgo(date, {
  short: true,
  maxUnits: 2,
})
// "2h 30m ago"
```
